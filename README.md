# Infinity Tabs 2020

A Windows batch script that opens infinite browser tabs and command prompts until your device cannot handle it.

## Description

Infinity Tabs 2020 is a simple yet effective Windows batch script that demonstrates the concept of infinite loops and system resource consumption. The script continuously opens Chrome browser windows, Internet Explorer windows, and Command Prompt instances in an endless loop, designed to stress-test or crash a system by exhausting its resources.

## Features

- Opens Chrome browser windows infinitely
- Opens Internet Explorer windows infinitely
- Spawns Command Prompt instances continuously
- Uses a simple infinite loop mechanism in batch scripting
- Lightweight single-file implementation

## Technologies Used

- Windows Batch Script (.bat)
- Windows Command Prompt
- Chrome Browser
- Internet Explorer

## Installation

```shell
# Clone the repository
git clone https://github.com/bryanseah234/infinityTabs2020.git

# Navigate to project directory
cd infinityTabs2020
```

No additional dependencies required - just Windows OS with Chrome and Internet Explorer installed.

## Usage

```cmd
:: Double-click the batch file to run
infinite tabs crash.bat
```

**Alternative method:**
1. Download the repository as a zip file
2. Unzip the file
3. Double-click the `infinite tabs crash.bat` file to run the program

**⚠️ WARNING:** This script will open windows indefinitely until manually stopped or the system runs out of resources. Use Task Manager (Ctrl+Alt+Delete) to end the processes if needed.
